# Module: terminal

Installs the Kitty terminal emulator with a themed configuration.

## Packages

| Package | Manager | Purpose |
|---|---|---|
| `kitty` | AUR | GPU-accelerated terminal emulator |

## Dotfiles

| Source | Target |
|---|---|
| `dotfiles/terminal/kitty.conf` | `~/.config/kitty/kitty.conf` |

## Verification

Checks `kitty --version` succeeds.

## Dependencies

- `core`
