# Module: waybar

Installs the Waybar status bar with a custom configuration tailored for Hyprland.

## Packages

| Package | Manager | Purpose |
|---|---|---|
| `waybar` | pacman | Highly customisable status bar |

## Dotfiles

| Source | Target |
|---|---|
| `dotfiles/waybar/config.jsonc` | `~/.config/waybar/config.jsonc` |
| `dotfiles/waybar/style.css` | `~/.config/waybar/style.css` |

## Verification

Checks `waybar` is present in `$PATH`.

## Supported Architectures

- `x86_64` only

## Dependencies

- `core`
- `hyprland`
